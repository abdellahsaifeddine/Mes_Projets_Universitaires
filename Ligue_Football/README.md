# ⚽ Système de Gestion d'une Ligue de Football

## 📖 Contexte du Projet
Projet universitaire réalisé dans le cadre du module de Base de Données. Gestion complète d'une ligue de football professionnelle : équipes, joueurs, matchs, classements, statistiques.

## 🛠️ Technologies et Outils
* SGBD : Microsoft Access
* Langage : SQL
* Conception : Modèle Entité-Association, Modèle Relationnel (Normalisation)

## 🎯 Fonctionnalités
* Gestion des entités sportives : clubs, joueurs, arbitres
* Suivi des rencontres : scores, événements (buts, cartons)
* Statistiques et classements automatiques via SQL

## 📂 Structure
* /Documentation : modèles conceptuels et rapports PDF
* /Base_Access : fichier .accdb de la base finale
* /Scripts_SQL : code SQL extrait de Access
